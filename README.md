**วิธี install พิมพ์ลง command prompt ทีละบรรทัด**  
1.git clone https://github.com/fufu-2345/webFramework.git
2.cd webFramework
3.setup  
4.Try npm start

---

**start ได้ 2 วิธี**  

- Production Build :  
  npm run build ////ก่อนถึงจะ  
  npm start ////ได้ โดยหากไม่มี .next ต้องทำตามนี้เพื่อ

- Development Mode :  
  npm run dev //// start ได้เลย ต่อให้ไม่มี .next แต่ถ้าทำบ่อยๆ cache เยอะ เปลือง
